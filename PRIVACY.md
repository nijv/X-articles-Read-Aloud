# Privacy Policy for X Articles Read Aloud

**Last Updated:** January 11, 2026

## Overview

X Articles Read Aloud is committed to protecting your privacy. This extension operates entirely on your local device and only communicates with OpenAI's API for text-to-speech generation.

## Data Collection

### What We Collect
- **OpenAI API Key**: Stored locally in your browser using Chrome's storage API
- **User Preferences**: Voice selection and playback settings stored locally

### What We DON'T Collect
- We do NOT collect any personal information
- We do NOT track your browsing history
- We do NOT store or transmit article content
- We do NOT use analytics or tracking services
- We do NOT share any data with third parties (except OpenAI for TTS generation)

## Data Usage

### OpenAI API Communication
When you click "Read Aloud" on an X article:
1. The article text is extracted from the page
2. The text is sent to OpenAI's API using YOUR API key
3. OpenAI generates audio and returns it to your browser
4. The audio is played locally - nothing is stored

This communication is governed by [OpenAI's Privacy Policy](https://openai.com/policies/privacy-policy).

### Local Storage
- Your API key is stored in Chrome's local storage
- Voice preferences are stored locally
- All data remains on your device and is never transmitted to our servers

## Permissions Explained

The extension requests the following permissions:

- **activeTab**: To detect when you're on an X article page
- **storage**: To save your API key and preferences locally
- **scripting**: To inject the "Read Aloud" button on article pages
- **Host permissions (x.com, twitter.com)**: To work on X/Twitter article pages
- **Host permissions (api.openai.com)**: To communicate with OpenAI's API

## Third-Party Services

### OpenAI
This extension uses OpenAI's Text-to-Speech API. When you use the extension:
- Article text is sent to OpenAI for audio generation
- OpenAI processes the request according to their privacy policy
- Audio is returned to your browser

Please review [OpenAI's Privacy Policy](https://openai.com/policies/privacy-policy) for details on how they handle data.

## Data Security

- Your API key is stored securely using Chrome's storage API
- All communication with OpenAI uses HTTPS encryption
- No data is stored on external servers

## Your Rights

You can:
- Delete all stored data by removing the extension
- Clear your API key at any time through the extension popup
- Stop using the extension at any time

## Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect information from children.
